Como configurar
Siga os passos abaixo na primeira vez
1
Faça login com a conta Microsoft
Clique em "Entrar com Microsoft" acima. Use a mesma conta do OneDrive onde o arquivo Excel está salvo (conta pessoal @outlook.com ou @hotmail.com).
2
Autorize o acesso ao OneDrive
Uma janela da Microsoft abrirá pedindo permissão. Clique em "Sim" ou "Aceitar". O app só acessa arquivos Excel — nenhum outro dado.
Se aparecer aviso de "app não verificado", clique em "Informações avançadas" → "Continuar assim mesmo". Isso ocorre porque o app ainda não passou pela verificação formal da Microsoft (processo pago).
3
Suba o arquivo Excel para o OneDrive
No OneDrive, crie uma pasta Documentos e faça upload do arquivo Diario_Pedagogico_Musical.xlsx que foi gerado.
No tablet: abra o app OneDrive → toque em + → Carregar → selecione o arquivo Excel.
4
Selecione o arquivo no app
Após o login, use o campo "Caminho no OneDrive" acima e clique em "Buscar". Se o caminho padrão não funcionar, ajuste conforme onde salvou o arquivo.
5
Primeira sincronização
Clique em "Sincronizar agora". O app lerá os dados do Excel e estará pronto para uso. A partir daí, cada lançamento é salvo automaticamente.
ID do aplicativo (Azure)
Necessário para autenticação com a Microsoft
Para usar o app em produção, você (ou um desenvolvedor) precisa registrar o app no portal.azure.com e colar o ID aqui. Veja o README completo para o passo a passo com prints.
Ver instruções de registro no Azure
1
Acesse portal.azure.com
Entre com a conta Microsoft. Vá em Azure Active Directory → Registros de aplicativo → Novo registro.
2
Configure o registro
# Preencha assim: Nome: Diário Pedagógico Musical Tipos de conta: Contas pessoais Microsoft URI de redirecionamento: https://SEU-USUARIO.github.io/REPO/
3
Copie o ID e cole acima
Após criar, copie o ID do aplicativo (cliente) e cole no campo acima.
4
Permissões necessárias
